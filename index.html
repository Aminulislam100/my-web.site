HTML

HTML (HyperText Markup Language) is used to create the structure of web pages. It organizes content with headings, paragraphs, images, and links. Every website relies on HTML as its foundation.

CSS

CSS (Cascading Style Sheets) adds style to web pages. It controls colors, fonts, layouts, and spacing. Without CSS, websites look plain and incomplete.

Java Script

JavaScript makes websites interactive. It runs in browsers or on servers with Node.js and adds features like animations, forms, and dynamic updates. Modern JS includes classes, arrow functions, and modules.

React

React is a JavaScript library for building web interfaces. It uses reusable components and efficiently updates the page with the virtual DOM. React helps create fast and interactive web apps.
